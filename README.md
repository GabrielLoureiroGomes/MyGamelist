## :boom: How to run

- ### **Prerequisites**

    - Make sure you have JDK version 11 in your `PATH`

## :hammer: Application

### Clone the repository:

```sh
# HTTPS
  $ https://github.com/GabrielLoureiroGomes/MyGamelist.git
```

```sh
# SSH
  $ git@github.com:GabrielLoureiroGomes/MyGamelist.git
```

### Build the application

```sh
  $ gradle build
```

### Run the application

```sh
  $ gradle run
```
